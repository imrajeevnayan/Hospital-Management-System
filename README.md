```bash

cd backend && ./mvnw spring-boot:run

# 2. Open the UI
# Just double-click frontend/index.html — That's it!
```

## 📖 Easy Step-by-Step Guide

Setting up HealthSync for the first time? Follow these simple steps.

### 1️⃣ Prepare your Database
You need a place to store patient data. 
*   Open your MySQL tool (like MySQL Workbench or Terminal).
*   Run: `CREATE DATABASE hms;`
*   Open `backend/src/main/resources/application.yml` and update your username/password.

### 2️⃣ Fire up the Backend 🚀
The "Brain" of the project runs on Spring Boot.
*   Open a terminal/command prompt.
*   Type `cd backend`
*   Type `./mvnw spring-boot:run` (Mac/Linux) or `mvnw.cmd spring-boot:run` (Windows).
*   **Success looks like:** You see `Started HospitalManagementSystemApplication in X seconds` in your terminal.

### 3️⃣ Open the Dashboard 🎨
The "Beauty" of the project is in the frontend.
*   Navigate to the `frontend` folder.
*   Right-click `index.html` and select **"Open with Chrome"** (or any modern browser).
*   No installations, no npm, no waiting. It just works!
